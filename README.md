Barcode bot

1. Установка зависимостей
```bash
pip install -r requirements.txt
```

2. Настройка config
```
в config.py заполняем BOT_TOKEN и USERS_WHITELIST
(свой id можно получить с помощью команды /recipient)
```

3. Запуск бота
```bash
python main.py
```
